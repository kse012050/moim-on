모임Talk 보안 시스템

  • 보안정책
  • 통신수간 암호화
  • 데이터 암호화

차이니즈월(Chinese Wall) 적용을 통한 부서 간 통신 제약 및 보안 강화

차이니즈월이란, 기업 내 정보교류를 차단하는 장치로
특히 보안이 중요한 금융기업에서 필수적으로 부서 간 또는 계열사 간의 정보 교류를 차단을 위해 활용합니다.

SSL/TLS 적용으로 서버와 클라이언트간 데이터를 안전하게 송/수신

SSL/TLS 는 웹사이트/브라우저/서버 간 전송된 데이터를 암호화하여 보안을 유지하는 표준 기술로,
개인 정보 및 금융 정보를 포함한 전송되는 정보의 해킹을 방지합니다.

MoimTalk 설치형은 Proxy 역할을 하는 Nginx 웹서버에 인증서 적용하는 것을 기본으로 진행하며,
고객사의 표준 웹 서버 사용 요청 시 요청 여건에 맞춰 변경 설치를 진행합니다.

PostgreSQL 를 통한 강력한 데이터 암호화

일반 텍스트 또는 암호화되지 않은 버전의 데이터를 암호문 또는 암호화된 버전으로 변환하는 방식으로
해독 키를 소유한 사용자만 데이터에 액세스할 수 있습니다

MoimTalk 설치형은 PostgreSQL 데이터베이스를 표준으로 사용하며
고객사의 표준 데이터베이스 또는 DB암호화 솔루션 연계 요청 시 적용하여 설치합니다
