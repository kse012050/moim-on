빠른 현업이 가능한
조직도 기반 메신저

  1. 프로젝트에 맞춘 대화방 설정

    조직도를 이용한 대화방 개설, 개인 프로파일에서 대화 요청
    “1:1, 1:N, 그룹(팀), 프로젝트별 대화

  2. 기업용 메신저에 최적화

    인사 시스템과 연계한 조직도 구성
    단일로그인(SSO), LDAP, AD등 연계 로그인
    PC/모바일 앱 배포 관리

  3. 조직도기반의 편의 기능

    즐겨찾기 추가, 개인그룹 추가
    메일 보내기, 채팅, 구성원 초성 검색

다양한 자료 공유와
유연한 커뮤니케이션

  1. 대화방에서

    대화상대 초대, 알림끄기, 대화내용 저장
    파일 모아보기 대화방 나가기 등 기능 제공

  2. 실시간 확인, @멘션, 이모지

    빠른 의사결정이 가능한 실시간 확인 기능
    @멘션으로 다수 멤버 중 특정인을 지정하며
    이모지 추가로 감성을 전달

필요에 따라
가장 빠르게 접근하는
데이터 검색

  1. 대화방 내 저장된 파일 필터링

    미디어 탭에서 이미지, 동영상등 멀티미디어 파일 검색
    파일 탭에서 오피스 파일등 문서 파일 검색
    링크 탭에서 URL 링크 정보 검색

  2. 전체 검색

    현재 조회 중인 협업 대화방 기준으로 검색
    단계별 필터링으로 검색 범위 조정
    컨텐츠 내용과 파일명으로 검색

기업에 가장 적합한
사내시스템 연계

  1. 메신저로 공유

    그룹웨에등 기간계 시스템에서
    공유하고자 하는 컨텐츠를 대상으로
    메신저 대화방에 바로가기 등록

  2. 메일 발송 및 메뉴 연계

    메신저 더보기 메뉴에 기간계 시스템 바로가기 구성
    메신저내 조직도 또는 사용자 프로파일 조회 중
    선택된 사용자를 수신인으로 메일 작성

  3. 알림봇

    그룹웨에, ERP, SCM등
    주요 기간계 시스템 연계한 알림 수신 API 제공
    단일로그인(SSO) 연계 시 바로가기 지원

재택근무에 적합한
온라인 대면 회의

  1. PC와 모바일 화상회의

    PC 다자간 화상회의
    모바일 참석자 화면 분할
    모바일 사용자간 화상회의

  2. 화상회의 개설 및 일정 연계

    화상회의 개설 시 일정관리 연계
    일정 도래 알림 시 화상회의 알림

외부 사용자 초대로
더 강력해진 협업 기능

  1. 외부 사용자 초대 및 참여

    외부 협력 인원에게 초대 메일 발송,
    지정된 프로젝트 또는 협업대화방에 참여

  2. 메일 발송

    메일을 연계하여 조직도나 개인 프로파일에서
    단일 또는 다수의 수신자를 지정해서 메일 작성

PC/모바일 완벽한 호환으로
업무 연계성 강화

  1. 모바일 앱으로 활용도 향상

    협업 공간 생성, 참여, 활동 일련의 작업은
    모바일에서 동일하게 제공

  2. 친숙한 UI/UX

    누구에게나 친숙한 UI/UX
    모바일과 PC 메신저 호환 상호 호출
